# Music Assets

## Purpose
Stores music asset files used by the game.

## Ownership
Owned by Content Pipeline + UI/Presentation tracks (Phase 4.95 Architecture Lock).

## Related Systems
Loaded by UI rendering, map/event presentation, and audio playback systems.

## Future Expansion
Future phases add production-quality assets, import automation, and validation checks.

## File Naming Convention
Use lowercase snake_case names. Follow domain-specific conventions where defined (e.g., `flag_{nation_id}.png`, `portrait_{leader_id}.png`).
