# ID Standard

Permanent ID naming convention:

- `nat_usa` -> Nation
- `prov_usa_dc` -> Province
- `county_usa_montgomery` -> County
- `city_usa_dayton` -> City
- `leader_george_washington` -> Leader
- `gov_constitutional_monarchy` -> Government
- `religion_catholic` -> Religion
- `party_democrat` -> Political Party
- `tech_industrialization` -> Technology
- `resource_oil` -> Resource
- `event_1939_poland` -> Event
- `company_ford` -> Company

IDs are lowercase snake_case with a stable prefix per domain.
