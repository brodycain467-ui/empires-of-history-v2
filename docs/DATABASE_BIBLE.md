# Database Bible

All records must support source tracking metadata:

```json
{
  "sources": ["Encyclopaedia Britannica", "Library of Congress"],
  "confidence": "Verified",
  "verified": true,
  "notes": ""
}
```

Database envelope standard:

```json
{
  "database": "",
  "schema_version": 1,
  "minimum_engine_version": "0.4.95",
  "last_updated": "",
  "record_count": 0,
  "records": []
}
```
