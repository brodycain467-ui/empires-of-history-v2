# Godot 4 Project Setup Guide\n\n## Initial Project Setup\n\n### 1. Create Godot 4 Project\n\n```bash\n# Create project.godot file\necho 'config_version=5' > project.godot\n```\n\n### 2. Project Settings (project.godot)\n\n```ini\n[application]\nconfig/name=\"Empires of History V2\"\nconfig/description=\"Grand Strategy Game - Data Driven Architecture\"\nconfig/version=\"0.1.0\"\nconfig/tags=PackedStringArray(\"strategy\", \"game\", \"history\")\n\n[input]\n\n[layer_names]\nrenderer_2d/layer_1=\"UI\"\nrenderer_2d/layer_2=\"Map\"\nrenderer_2d/layer_3=\"Background\"\n\n[rendering]\nrenderer/rendering_method=\"gl_compatibility\"\n\n[dotnet]\nproject/assembly_name=\"EmpiresOfHistory\"\n```\n\n### 3. C# Project Structure\n\nCreate `empires-of-history-v2.csproj`:\n\n```xml\n<Project Sdk=\"Microsoft.NET.Sdk\">\n  <PropertyGroup>\n    <TargetFramework>net8.0</TargetFramework>\n    <LangVersion>latest</LangVersion>\n    <Nullable>enable</Nullable>\n  </PropertyGroup>\n\n  <ItemGroup>\n    <PackageReference Include=\"xunit\" Version=\"2.6.0\" />\n    <PackageReference Include=\"xunit.runner.visualstudio\" Version=\"2.5.0\" />\n    <PackageReference Include=\"Microsoft.NET.Test.Sdk\" Version=\"17.7.0\" />\n  </ItemGroup>\n</Project>\n```\n\n### 4. Folder Structure Creation\n\n```bash\n# Create all folders\nmkdir -p src/Core\nmkdir -p src/Data/Schemas\nmkdir -p src/Data/Models\nmkdir -p src/Simulation\nmkdir -p src/UI/Scenes\nmkdir -p src/UI/Scripts\nmkdir -p src/UI/Managers\nmkdir -p src/Map/Scenes\nmkdir -p src/Map/Scripts\nmkdir -p src/Map/Managers\nmkdir -p src/Save\nmkdir -p src/History\nmkdir -p src/Utilities\nmkdir -p src/Godot\n\nmkdir -p assets/textures/ui\nmkdir -p assets/textures/map\nmkdir -p assets/textures/icons\nmkdir -p assets/fonts\nmkdir -p assets/themes\nmkdir -p assets/sounds/sfx\nmkdir -p assets/sounds/music\n\nmkdir -p data/provinces\nmkdir -p data/nations\nmkdir -p data/history\nmkdir -p data/config\nmkdir -p data/saves\n\nmkdir -p tests/Core\nmkdir -p tests/Data\nmkdir -p tests/Simulation\nmkdir -p tests/History\nmkdir -p tests/Save\n\nmkdir -p docs\nmkdir -p .github/workflows\n```\n\n### 5. AutoLoad Singletons\n\nCreate `src/Godot/GameManager.cs`:\n\n```csharp\nusing Godot;\n\nnamespace EmpiresOfHistory.Godot\n{\n    public partial class GameManager : Node\n    {\n        public static GameManager Instance { get; private set; }\n\n        public override void _Ready()\n        {\n            Instance = this;\n            GD.Print(\"GameManager initialized\");\n        }\n    }\n}\n```\n\nRegister in `project.godot`:\n\n```ini\n[autoload]\nGameManager=\"res://src/Godot/GameManager.cs\"\n```\n\n### 6. Build and Test\n\n```bash\n# Build C# project\ndotnet build\n\n# Run tests\ndotnet test\n```\n\n## Scenes Setup\n\n### Main Scene Hierarchy\n\n**Main.tscn:**\n```\nMain (Node2D)\n├── WorldMap tests/\n```\n\n### Test Structure\n\nEach test file follows XUnit pattern:\n\n```csharp\npublic class SystemTests\n{\n    [Fact]\n    public void Method_Condition_ExpectedResult()\n    {\n        // Arrange\n        // Act\n        // Assert\n    }\n}\n```\n\n## Development Workflow\n\n1. **Define data schema** in JSON files\n2. **Create C# model classes** to represent data\n3. **Implement DataLoader** to load JSON\n4. **Write unit tests** in tests/ folder\n5. **Create Godot scenes** in src/*/Scenes/\n6. **Attach C# scripts** as scene node scripts\n7. **Test in Godot editor**\n8. **Iterate and refine**\n\n## Key Development Tips\n\n### C# in Godot\n\n- All public methods are called by Godot: `_Ready()`, `_Process()`, `_Input()`\n- Use `partial` for script classes\n- Ensure classes inherit from appropriate Godot types\n- Use `[Export]` for inspector variables\n\n### JSON in Godot\n\n- Use `Json` class to parse: `var json = new Json(); json.Parse(content);`\n- Validate against schemas in C#\n- Store in `/data` folder\n- Load via DataLoader\n\n### Performance\n\n- Use object pooling for thousands of provinces\n- Lazy load historical data\n- Cache frequently accessed data\n- Use viewport culling for off-screen elements\n\n## Build for Platforms\n\n### iOS/iPad\n\n```bash\ngodot --export-release iOS\n```\n\n### Android\n\n```bash\ngodot --export-release Android\n```\n\n### PC (Linux/Windows/Mac)\n\n```bash\ngodot --export-release Linux\n```\n\n## Debugging\n\n### Print Statements\n\n```csharp\nGD.Print(\"Debug message\");\nGD.PrintErr(\"Error
