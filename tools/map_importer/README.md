# Map Importer Tool

## Purpose
Import map geometry/metadata from external formats.

## Ownership
Owned by Tools and Pipeline track (Phase 4.95 Architecture Lock).

## Related Systems
Produces artifacts consumed by runtime data loaders and simulation/event/map systems.

## Future Expansion
Implemented in later phases with CLI/GUI workflows, validation, and automation.

## File Naming Convention
Source files use lowercase snake_case by default; binaries/artifacts include tool name and semantic version.

## Expected Inputs/Outputs
- Inputs: structured project data and configuration files
- Outputs: validated/transformed JSON or map/editor artifacts
- Planned implementation phase: post-Phase 5 tooling track
